## 安装

```bash
npm install @wont/biz-ui --save
```

yarn add @wont/biz-ui
```

pnpm add @wont/biz-ui
```

如果你的网络环境不佳，推荐使用 [nrm](https://github.com/Pana/nrm)切换源
